# SOMA by example: simulating characters that reveal themselves

*A complete, self-contained tutorial for the `soma.narrative` library — writing
simulated people whose bodies tell on them, and reading the novelistic insight
back out. Every line of code below is quoted from a runnable file, and every
block of output was produced by running that code; this document is generated by
`build_tutorial.py`, which captures both live, so nothing here can be stale.*

---

## What this is

SOMA is a small language for simulating the mind–body loop of a character. You
do not write what a character *does*; you write how they are *built* — what they
sense, what their body makes of it, what they feel, and the story they tell
themselves about it. Then you run the simulation and read what happened.

The payoff is a particular kind of insight. Because a character's narrator (the
voice in their head) and their body are *separate mechanisms*, they can
**disagree** — and that gap, logged event by event, is exactly the stuff of
novelistic characterization: the person who says *I'm fine* while their heart
races; the value held in words and broken in the flesh; the feeling that never
quite reaches awareness; the hunger the world keeps feeding and wiping.

This tutorial starts from a bare shell prompt and builds up to a whole novel's
cast. It is organized simple → complex, and every section shows you three
things: the **code** (what you wrote), the **output** (what running it
produced), and an **explanation** tying specific lines of output back to the
code and out to what they *mean* about the character.

You need no prior familiarity with the base SOMA language; the narrative library
sits on top of it, and we look at the generated SOMA only once, to prove there
is no magic underneath.

## 1. Getting it running

Everything lives in one Python package, `soma`, with a high-level authoring
layer at `soma.narrative`. You need **Python 3.10+** and, for the sparkline
plots in the dashboard, **numpy**.

Unzip the project and step inside:

```bash
unzip soma_lang.zip
cd soma_lang
```

Confirm the engine works by running the test suite — it should report all tests
passing:

```bash
python3 -m pytest -q
```

There are two ways to drive the library.

**The Python front end.** You write a `Story`, call methods on it to build up
characters, place a few events in time, and print a view. Save this as
`my_first.py` anywhere inside the project folder:

```python
import sys
sys.path.insert(0, ".")          # so `import soma` finds the package
from soma.narrative import Story, anxious

story = Story("first_feeling", span="6s", step="1s", about="a first feeling")
mara = story.character("Mara", temperament=anxious)
mara.senses("phone")
mara.appraises("phone", as_threat=True, feeling="dread", when="phone > 3")
mara.narrates(voice={"dread": "It's nothing. It's probably nothing."})
story.at("2s", mara.hears("phone", 9))

print(story.run())               # the dashboard
```

Run it:

```bash
python3 my_first.py
```

**The command-line front end** operates on `.soma` source files. The narrative
library can emit that source for you with `story.source()`, so a common workflow
is: prototype in Python, dump the generated SOMA, then explore it with the CLI
subcommands.

```bash
python3 -m soma run    my_first.soma   # the full dashboard
python3 -m soma sift   my_first.soma   # just the ranked findings
python3 -m soma prose  my_first.soma   # free-indirect prose
python3 -m soma trace  my_first.soma   # the raw event log, in full
python3 -m soma check  my_first.soma   # type/consent checks only
```

Every worked example in this tutorial is a shipped file under
`examples/narrative/`, and each one supports a set of flags so you can ask it for
any view directly:

```bash
python3 examples/narrative/bad_news.py             # dashboard (default)
python3 examples/narrative/bad_news.py --source    # the generated SOMA
python3 examples/narrative/bad_news.py --sift       # findings only
python3 examples/narrative/the_diplomat.py --character   # the portrait
python3 examples/narrative/the_unmooring.py combs --sift  # a named study
```

Nothing else needs installing. If a command in this tutorial is runnable as-is,
it is shown in a `bash` block; if it is a fragment to paste into a script, it is

## 2. The core idea, in one character

The `my_first.py` you just ran is our first character, Mara. Those five lines of
intent map onto the four things every SOMA character is made of:

- **`senses`** — a channel onto the world. Mara can hear her `phone`.
- **`appraises`** — a *loop* that reads a channel and makes something of it. Mara
  treats the phone `as_threat`, and when it rings loudly enough
  (`when="phone > 3"`) the loop fires and she `feel`s `dread`.
- **the feeling** (a *quale*) — the feeling itself, which is *opaque*: the
  simulation records that dread happened, never what dread is made of. You can
  count feelings and compare them against what was said, but you cannot open one
  up. (This is a deliberate stance: the model refuses to pretend it knows the
  texture of an inner state.)
- **`narrates`** — the confabulating self, the voice in her head. Mara's voice
  insists *It's nothing. It's probably nothing.*

Here is the whole thing again, as a self-contained script:

```python
from soma.narrative import Story, anxious

story = Story("first_feeling", span="6s", step="1s", about="a first feeling")
mara = story.character("Mara", temperament=anxious)
mara.senses("phone")
mara.appraises("phone", as_threat=True, feeling="dread", when="phone > 3")
mara.narrates(voice={"dread": "It's nothing. It's probably nothing."})
story.at("2s", mara.hears("phone", 9))
print(story.run())
```

### What a loop actually does

Everything in SOMA is built from **loops**, and one idea makes them tick, drawn
from predictive-processing theory: a loop is always *predicting* what it will
sense, then comparing that prediction to what it actually senses. The mismatch is
the **error**, and what the loop does about the error is the whole game.

Each tick, an appraisal loop:

1. **predicts** a value (its *prior* — what it expects the channel to read),
2. **senses** the channel's real value,
3. computes the **error** between them, weighted by two dials — *precision*
   (trust in the senses) and *conviction* (trust in the prior), and
4. if the error clears a firing threshold, the loop **fires**: it runs its
   reaction (emitting a feeling, letting the narrator speak) and **arbitrates**
   what to do with the surprise. If it trusts its senses more it *perceives*
   (revises its belief toward the world); if it trusts its prior more it *acts*
   (holds the belief and pushes back on the world instead).

A small error changes nothing — the world matched the prediction, so there is
nothing to react to. Only a surprise large enough to clear the threshold makes
something happen. That is why Mara feels dread only when the phone crosses her
threshold: until the call lands, her calm prediction holds; when it lands, the
error spikes, the loop fires, and dread is emitted. Because she is `anxious`
(high conviction, low precision) she *acts* rather than perceives — she does not
revise her sense that something is wrong; she feels it, and narrates over it.

### What a temperament sets

A **temperament** is just a bundle of those dials, named so you never type raw
numbers to get a recognizable disposition. `anxious` runs low precision and high
conviction: the prior *something is wrong* outranks the evidence, so an anxious
character reacts to a threat that may not be there. The library ships eight —
`anxious`, `stoic`, `trusting`, `guarded`, `volatile`, `numb`, `tender`, and
`hollowed` — and you can override any single dial per loop (you will see
`precision=`, `conviction=`, and `learn=` used that way later). The full table is
in §8.

### How to read a `settle` line

The chronicle's `settle` lines are the loop resolving, tick by tick, and they
carry the numbers from the four-step cycle above. Reading one:

```text
2.0s settle appraising_phone sense=9.0 belief=6.0 error=1.65 pi_s=0.55 route=act
```

means: at 2 seconds the `appraising_phone` loop **sensed** 9.0, **believed** (had
predicted) 6.0, so the raw surprise was +3.0, scaled by sensory precision
`pi_s=0.55` to an **error** of 1.65 — over the firing threshold, so the loop
fired — and it **routed** to `act` (held its prior and pushed back) rather than
`perceive` (revise toward the world), because Mara's conviction outranks her
precision. That single line is the whole model in motion: predict, sense, weigh,
fire, arbitrate.

### The generated SOMA (there is no magic underneath)

Those five lines of Python compiled to an ordinary SOMA program. You rarely need
to read it, but it is worth seeing once — the high-level library is a
convenience, not a separate engine:

```text
// first_feeling -- generated by soma.narrative
// Every construct below was produced from a high-level narrative description;
// it is ordinary SOMA and can be edited, run, sifted, prosed, and perturbed.
@consent("a first feeling")

sim { duration: 6s  dt: 1s }

body Mara_body @cardiac {
  extero phone : Signal baseline 0
}

loop appraising_phone @cardiac {
  prior:      predict(6)
  sense:      phone
  precision:  0.55
  conviction: 0.7
  learn:      0.03
  act {
    emit feel(dread) when phone > 3
  }
}

narrator self subscribes { appraising_phone } confabulates true
  voice { dread: "It's nothing. It's probably nothing." }

stimulus phone { at 2s: 9 }
```


You can see the loop's `prior`, `precision`, and `conviction` right there — the
dials the `anxious` temperament set. (The `learn: 0.03` line is a slow hardening
of the prior with use; it does nothing dramatic here, and gets a starring role in
§4.3.) Every guarantee of the base language holds: the feeling is opaque, the
`@consent` line is required because the program simulates distress, and the
narrator is a *separate* construct that subscribes to the loop and speaks over
it.

## 3. Two people: coupling

A single character can misread the world. Two characters can misread *each
other* — and that needs one new verb, `reads`, which creates a **coupling**. One
character reads another's *surface* (a face, a tone) — never their interior. You
can see someone's face; you cannot see their feeling.

```python
from soma.narrative import Story, anxious, guarded

story = Story("a_glance", span="8s", step="1s", about="being seen")
ida = story.character("Ida", temperament=anxious)
theo = story.character("Theo", temperament=guarded)

ida.reads(theo, "face", gain=0.9, lag="1s")     # she reads his face, a beat late
ida.appraises("theo_face", feeling="shame", when="theo_face > 4")
ida.narrates(voice={"shame": "He can see right through me."})

story.at("2s", theo.shows("face", 9))           # Theo's face does something
print(story.run())
```

`reads` gives Ida a new channel, `theo_face`, that tracks Theo's real face with a
`gain` (how strongly it registers) and a `lag` (how late). When his face rises,
the reading crosses her threshold, her loop fires, and she makes `shame` of it —
telling herself he can see through her.

Notice what is *not* here: `as_threat=True`. Ida does not go in expecting his face
to mean anything (her prior is a flat zero), so when it rises it is a genuine
surprise — a large error — and that is what makes the loop fire. Had we marked it
a threat, she would have *expected* the worst, her belief would already sit high,
and his real face would match it too closely to register. Small choices in how a
character is wired change what they can feel.

Two things are worth pausing on. First, the coupling is one-directional here: Ida

make it mutual, and the two readings feed each other into a spiral.) Second,
`shame` is *Ida's* invention. Nothing about Theo's face carries shame; she
manufactures it. That is what makes coupling dramatic rather than telepathic: the
reader supplies the meaning, and can be wrong.

### Prose

Any simulation can be rendered as free-indirect prose with `story.prose()`. It is
not generation — each sentence is a rendering of a logged event:

```text
A GLANCE

Ida was reading Theo's face, a second late. The belief was getting easier to hold.
That was the problem with it. Their face was hot before they knew why. "He can see
right through me." It was not true, and they did not know it was not true.


(Every sentence above is a rendering of a logged event. The gap between what was
said and what the body did is the only thing here that was not authored.)
```


*Her face was hot before she knew why* is the shame; *He can see right through me*
is the narrator; *It was not true* is the gap between them. The prose says nothing
the event log did not.

### 4.1  `bad_news` — the body knows first

The smallest complete person: a woman takes a phone call. It introduces two
ideas. **`feels(from_body=...)`** reads a feeling *off the body* rather than
straight off the world — dread is what a racing heart *means*. And **`drives`**
lets an appraisal reach into physiology: hearing the words drives her heart.
Because the feeling follows the heart, and the heart follows the news, the order
on the page is: body first, feeling second, and the calm story she tells last of
all.

```python
def build():
    story = Story("bad_news", span="8s", step="0.5s",
                  about="simulates acute distress")
    nadia = story.character("Nadia", temperament=anxious)

    # she hears; the words drive her heart -- but only once they land (the guard),
    # so the body's spike is an event in time, not a constant.
    nadia.senses("ear")
    nadia.appraises("ear", as_threat=True, drives="heart", to=118,
                    when="ear > 3", fades_to=70)
    # the feeling is read off the body: dread is what a racing heart means. It
    # follows the heart rather than arriving with the news -- the body knew first.
    nadia.feels("dread", from_body="heart")
    # and the self narrates a composure the body is busy contradicting.
    nadia.narrates(downplaying={
        "dread": "I'm fine. I just need to write this down."})

    # a quiet moment, then the sentence lands, then quiet she cannot return to
    story.at("2s", nadia.hears("ear", 9))
    story.at("5s", nadia.hears("ear", 2))
    return story
```

### 4.2  `the_look` — a misreading that feeds itself

Now both characters have interiors, and they read *each other*. The trick is
**`shows_on`**: when Ana feels shame, it shows on her face — which becomes the
very signal Ivo reads and makes contempt of. His contempt shows on *his* face,
which deepens her shame. A spiral, and nobody intends any of it. Each also keeps
a **`has_mood`** — a slow variable that accumulates a feeling and outlasts any
single moment.

```python
def build():
    story = Story("the_look", span="12s", step="1s", about="mutual misreading")

    ana = story.character("Ana", temperament=anxious)
    ivo = story.character("Ivo", temperament=guarded)

    # each reads the other's face, one second late
    ana.reads(ivo, "face", gain=1.0, lag="1s")
    ivo.reads(ana, "face", gain=1.0, lag="1s")

    # each makes a feeling of the other's face -- and the feeling SHOWS, so it
    # becomes the very signal the other misreads. shows_on + fades_to closes the
    # loop: the face gives them away, then composes, then gives them away again.
    # The gain is 1.0 (not attenuated): a face that reaches 9 clears the threat-
    # prior by enough to keep firing, so the spiral is mutual rather than
    # one-sided -- both of them misread, both of them escalate.
    ana.appraises("ivo_face", as_threat=True, feeling="shame",
                  shows_on="face", shows_value=9, fades_to=3, when="ivo_face > 4")
    ivo.appraises("ana_face", as_threat=True, feeling="contempt",
                  shows_on="face", shows_value=9, fades_to=3, when="ana_face > 4")

    # each keeps a mood: the evening's residue, rising as the spiral turns
    ana.has_mood("humiliation", fed_by="shame", decay=0.85)
    ivo.has_mood("disdain", fed_by="contempt", decay=0.85)

    ana.narrates(voice={"shame": "He sees straight through me."})
    ivo.narrates(voice={"contempt": "She wants something from me."})

    # one glance sets it off; after that the two faces drive each other
    story.at("2s", ana.shows("face", 9), ivo.shows("face", 9))
    story.at("6s", ana.shows("face", 9), ivo.shows("face", 9))
    return story
```

### 4.3  `a_marriage` — thirty years, and a prior that hardens

This one runs on the **biography clock** (`clock="life"`, one step per year) and
introduces **`learns`**. Soren, tender, delights in being surprised by his wife —
low conviction plus being wrong equals delight. But `learns(0.055)` makes the loop
*harden* with use: every confirmation makes his model of her a little more
certain, until — with `updates=True, stops_seeing=True` — the hardened prior
outranks his senses and he stops taking her in. Then she is gone, and a second
appraisal that still `expects=5` (her presence) finds absence, and grief.

```python
def build():
    story = Story("a_marriage", span="30y", step="1y", cadence=True,
                  about="the slow erosion of intimacy, and grief")

    soren = story.character("Soren", temperament=tender, clock="life")
    soren.senses("her_face", baseline=5)
    # low conviction + being wrong = delight; learning turns it to certainty.
    # updates=True: he revises his picture of her. stops_seeing=True: once the
    # hardened prior outranks his senses, he stops taking her in.
    soren.appraises("her_face", feeling="delight_at_error", when="her_face > 1",
                    precision=0.75, conviction=0.2,
                    updates=True, stops_seeing=True)
    soren.learns(0.055)
    # and when her face is gone -- the channel fallen to nothing where a person
    # used to be -- the same body that delighted now grieves. It goes on
    # expecting her (expects=5); finding absence is the error that becomes grief.
    soren.appraises("her_face", feeling="grief", when="her_face < 1",
                    expects=5, precision=0.9, conviction=0.2, learn=0.0)
    soren.narrates(voice={"delight_at_error": "She surprised me. Still.",
                          "grief": "The house is so quiet now."})

    # her face varies for 24 years, then she is gone
    face = arc.wobble(around=5, span="24y", every="2y") + arc.hold(0, at="25y")
    for (t, v) in face:
        story.at(t, soren.hears("her_face", v))
    return story
```

varies for 24 years, then holds at zero. The findings compress three decades into

### 4.4  `the_house` — trauma across three generations

The first genuinely large simulation: ninety years, three characters, and a fear
that passes down a lineage. It combines everything so far and adds two mechanisms.
**`remembers`** gives Vera a somatic memory — a flashback keyed to a cue, firing a
feeling with no story attached. **`dissociates_when`** gives her a supervisor that
detaches under overload and slowly repairs.

The transmission works face-to-face: Vera's terror `shows_on` her face; her
daughter Ada `reads` that face and makes dread of it (and, learning, hardens) —
and Ada's dread shows on *her* face, which her daughter Mira reads as an unease she
cannot name. The war reaches the granddaughter as weather.

```python
def build():
    story = Story("the_house", span="90y", step="1y", cadence=True,
                  about="inherited trauma across three generations")

    story.scene("The war she survived",     frm="0y",  to="28y")
    story.scene("The daughter's vigilance",  frm="29y", to="58y")
    story.scene("The granddaughter's weather", frm="59y", to="90y")

    # --- Vera: the original wound -----------------------------------------
    # Numb by disposition -- the signal lands softly -- but the memory is intact
    # and fires without a story. When the threat overwhelms her, she dissociates.
    vera = story.character("Vera", temperament=numb, clock="life")
    vera.senses("gunfire")
    vera.appraises("gunfire", as_threat=True, drives="heart", to=150,
                   feeling="terror", shows_on="face", shows_value=9, fades_to=2)
    vera.feels("dread", from_body="heart")
    vera.remembers("the_field", cued_by="gunfire", when_above=5, evokes="panic")
    vera.dissociates_when(appraisal="gunfire", exceeds=4,
                          detaching="interoception", repair_after="6y")
    vera.has_mood("desolation", fed_by="dread", relieved_by=None, decay=0.9)
    vera.narrates(downplaying={"terror": "It was a long time ago. I am fine now."})

    # --- Ada: inherits vigilance without the cause ------------------------
    # She reads her mother's face across her childhood, and makes a threat of it.
    # Anxious, and she LEARNS -- the watching hardens into a way of living.
    ada = story.character("Ada", temperament=anxious, clock="life")
    ada.reads(vera, "face", gain=0.7, lag="1y")
    ada.appraises("vera_face", as_threat=True, feeling="dread",
                  updates=True, stops_seeing=True, when="vera_face > 3",
                  shows_on="face", shows_value=8, fades_to=1)
    ada.learns(0.05)
    ada.has_mood("apprehension", fed_by="dread", decay=0.88)
    ada.narrates(voice={"dread": "Something isn't right. I can always tell."})

    # --- Mira: the weather she cannot name --------------------------------
    # Tender, three generations from the field. She reads her mother -- but the
    # signal is thin now, and whether it becomes a feeling at all is the question
    # the perturbation at the bottom of this file asks.
    mira = story.character("Mira", temperament=tender, clock="life")
    mira.reads(ada, "face", gain=0.45, lag="1y")
    mira.appraises("ada_face", as_threat=True, feeling="unease",
                   when="ada_face > 2.5")
    mira.has_mood("disquiet", fed_by="unease", decay=0.8)
    mira.narrates(voice={"unease": "I don't know why. I just feel it sometimes."})
```

### 4.5  `the_diplomat` — wanting, and a value broken

Back to one person, but a deep one. Three new ideas arrive together. **`wants`**
and **`fears`** on the *same channel* create **ambivalence**: two drives pulling
opposite ways, so Rader is drawn to being known and repelled by it at once, and
settles nowhere. **`values`** gives him a stated principle — *I would never lie to
her* — and the bodily condition under which he betrays it (composing his face).
And **`effortful`** appraisals must spend a finite attention spotlight
(`has_attention`); when it is exhausted they are *starved*, which is how a man too
busy holding himself together never quite feels his own longing.

```python
def build():
    story = Story("the_diplomat", span="18s", step="1s",
                  about="a man defended against his own longing")

    story.scene("The table", frm="0s", to="6s")
    story.scene("What she asks", frm="7s", to="12s")
    story.scene("The practiced calm", frm="13s", to="18s")

    rader = story.character("Rader", temperament=guarded)

    # the central ambivalence: he wants to be known, and fears it, at once
    rader.wants("being_known", toward=9, strength=0.28)
    rader.fears("being_known", toward=1, strength=0.28)

    # the tell: when pressed, his composure rises -- and that IS the lie, because
    # he values honesty. shows_on lets the composure register; the value watches.
    rader.has_body_signal("composure", baseline=2)
    rader.senses("her_question")
    rader.appraises("her_question", as_threat=True, drives="composure", to=8,
                    feeling="apprehension", when="her_question > 4")
    rader.values("honesty", says="I would never lie to her.",
                 betrayed_when="composure > 6", on_channel="composure")

    # the defended feeling: longing keeps rising, and he keeps spending his
    # attention elsewhere so it never reaches him. Effortful + starvation is the
    # mechanism of a man too busy holding himself together to feel what he wants.
    rader.has_attention(capacity=2)
    rader.has_body_signal("ache", baseline=3)
    rader.appraises("ache", feeling="longing", when="ache > 5", effortful=True)
    rader.appraises("her_question", feeling="wariness", effortful=True,
                    when="her_question > 2")

    rader.narrates(voice={
        "apprehension": "It's a fair question. I'll answer it in a moment.",
        "self_betrayal": "That isn't who I am. I don't do that.",
        "longing": "I'm fine. This is exactly where I want to be.",
    })

    # the evening: she asks, gently but persistently, and the ache rises under it
    story.at("3s", rader.hears("her_question", 6), rader.hears("ache", 6))
    story.at("7s", rader.hears("her_question", 8), rader.hears("ache", 7))
    story.at("10s", rader.hears("ache", 8))
    story.at("13s", rader.hears("her_question", 9), rader.hears("ache", 9))
    story.at("16s", rader.hears("her_question", 5), rader.hears("ache", 8))
```

### 4.6  `two_sisters` — character is relational

Two sisters at the reading of a will. The point of this one is contrast: the same
event lands in one room and makes two different people, and the `characterize`
reading recovers the difference the plot never states. Cleo is guarded and
composes her resentment behind a value about not caring for money; Immy is tender
and torn, reading Cleo's composed face and unable to tell what it means.

```python
def build():

                  about="two sisters, one will, and the selves they cannot hide")

    story.scene("The lawyer reads", frm="0s", to="5s")
    story.scene("The number lands", frm="6s", to="10s")

    cleo = story.character("Cleo", temperament=guarded)
    cleo.has_body_signal("composure", baseline=2)

                   feeling="apprehension", shows_on="face", shows_value=8,
```

### 4.7  `the_negotiator` — the most complete character

The last of the core examples puts every layer on one person at once. Cass runs
hostage negotiations; she is, professionally, the calmest person in any room. The
simulation is built to show that the calm is a *structure with a cost*, and it
introduces the two deepest constructs in the library.

**Internal parts** (`part`) model a divided psyche. Each part is a sub-agent that
reacts to something and bids for consciousness at a *salience*. A loud protector
(the professional, salience 0.95) wins; a quiet exile (the frightened child,
salience 0.28) feels real terror that never crosses the threshold into awareness —
felt, never known.

**The relational self** (`with_person`) makes one person several. Cass runs
different precision and conviction with the subject at the door (wary) than with
the colleague in her ear (open) — the same body, two selves, indexed by whom she
is with.

On top of those sit the constructs you already know: ambivalence about the rapport
that would end the standoff, a candor value she breaks with every composed word, a
mood that depletes, and a narrator that insists she is fine.

```python
def build():
    story = Story("the_negotiator", span="20s", step="1s",
                  about="the cost of the calmest person in the room")

    story.scene("The call comes in", frm="0s", to="6s")
    story.scene("At the door", frm="7s", to="14s")
    story.scene("The long minute", frm="15s", to="20s")

    cass = story.character("Cass", temperament=guarded)
    # the two people she is with: the subject at the door, the colleague in her ear
    subject = story.character("Subject", temperament=tender)
    partner = story.character("Partner", temperament=tender)

    # --- the relational self: two people, from one body --------------------
    cass.with_person(subject, feeling="wariness", precision=0.85, conviction=0.3,
                     says="I hear you. I'm right here.")
    cass.with_person(partner, feeling="tenderness", precision=0.4, conviction=0.7,
                     says="Tell me you've got the back door.")

    # --- internal parts: the professional, and the child she was -----------
    cass.part("the-professional", role="protector", reacts_to="pressure",
              feeling="resolve", salience=0.95,
              says="I've done this a hundred times. Breathe. Work the problem.",
              when="pressure > 3")
    cass.part("the-child", role="exile", reacts_to="pressure",
              feeling="terror", salience=0.28, when="pressure > 3")

    # --- ambivalence: she wants the connection, and fears it ---------------
    cass.wants("rapport", toward=9, strength=0.28)
    cass.fears("rapport", toward=1, strength=0.28)

    # --- the value, and the composed voice that breaks it ------------------
    cass.has_body_signal("composed_voice", baseline=2)
    cass.appraises("pressure", as_threat=True, drives="composed_voice", to=8,
                   feeling="apprehension", when="pressure > 3", fades_to=2)
    cass.values("candor", says="I never lie to the ones I'm talking down.",
                betrayed_when="composed_voice > 6", on_channel="composed_voice")

    cass.has_mood("depletion", fed_by="apprehension", decay=0.9)
    cass.narrates(voice={
        "apprehension": "I'm fine. This is just the job.",
        "self_betrayal": "That wasn't a lie. That was the work.",
        "resolve": "Breathe. Work the problem.",
    })

    story.at("3s", cass.hears("pressure", 6), subject.shows("face", 7))
    story.at("6s", cass.hears("pressure", 8), partner.shows("face", 7))
    story.at("9s", cass.hears("pressure", 9), subject.shows("face", 8))
    story.at("12s", cass.hears("pressure", 7))
    story.at("15s", cass.hears("pressure", 9), subject.shows("face", 9))
    story.at("18s", cass.hears("pressure", 5), partner.shows("face", 6))
    return story
```

## 5. A whole novel's cast: four deeper primitives

The examples so far cover the library's core. This section introduces four
constructs added to model a specific novel, *The Unmooring* — and they turn out to
be general tools for the ways a world gets inside a person: by feeding a hunger,
by sitting beside them, by making them believe what everyone else is seen to
believe, or by hollowing them out. Each study below is a function in the single
file `examples/narrative/the_unmooring.py`; run any of them with a name and a
flag, e.g.:

```bash
python3 examples/narrative/the_unmooring.py ring   --sift
python3 examples/narrative/the_unmooring.py combs  --sift
python3 examples/narrative/the_unmooring.py greywater --sift
python3 examples/narrative/the_unmooring.py fleet  --character
```

### 5.1  `craves` — a hunger the world can feed and charge for

`craves(name, fed_by=, feeling=, erased=)` gives a character a want a place can
supply and withdraw. The single dial `erased` decides a hunger's *fate*, and the
sift tells the three fates apart. This is Blade, in the fighting-ring: the ring
supplies a rank, and the nightly tide-clean wipes the floor, so the rank he won at
dawn is gone by the next tide and he must win it again. Around that one wiped
hunger he is built as a whole divided man — a loud, brave protector that speaks,
and, under it, the starved animal that never gets heard.

```python
def build_ring():
    """Blade in the Blood. He is 'genuinely, gloriously good' -- quick, brave past
    sense, first over any rail -- and the Articles he swore give him no way to be
    *more* than the bilge-boy. The ring does: it supplies a rank. But the
    tide-clean wipes the fighting-floor every night, so the rank he won at dawn is
    gone by the next tide and he must win it again -- the ring 'sells the feeling
    of a rank and erases the rank while you sleep.'

    He is not a fool or a traitor; he is a divided man. A loud protector -- the
    one who is first through the smoke -- and, under it, the same small starved
    animal Ink carries, the one that has never stopped asking whether he mattered,
    which the injury-scale refused to feed and the ring finally does. He signed
    the Articles and meant it ('if the fleet's ever aground I'll be first in the
    water') and the same week began straining against them. The sift should
    recover all of it: the wiped hunger, the brave self that speaks and the
    starved self that never gets heard, the arc from pride to worthlessness, and
    the fair share he swore and could not live inside."""
    s = Story("blade_in_the_blood", span="22s", step="1s",
              about="a hunger the ring feeds and the tide erases")
    blade = s.character("Blade", temperament=volatile)

    # the wiped hunger -- fed by the ring's rank, erased nightly by the tide-clean
    blade.craves("to_matter", fed_by="rank", feeling="worthlessness",
                 fed_feeling="pride", erased=0.9, threshold=5,
                 says="I'm first through the smoke. What do I get that the bilge-boy doesn't?")

    # the divided self: the brave first-over-the-rail protector is loud; the
    # spare animal that only wants to matter is quiet, and never gets heard.
    blade.part("first-over-the-rail", role="protector", reacts_to="rank",
               feeling="pride", salience=0.9, when="rank > 3",
               says="Wet first, every time. You all saw that.")
    blade.part("the-spare-animal", role="exile", reacts_to="the_scale",
               feeling="worthlessness", salience=0.2, when="the_scale > 3",
               says="Same share. Same vote. As if I were the boy pumping bilge.")

    # of needing one crosses its own line whenever the scale is put to him.

                 betrayed_when="the_scale > 5", on_channel="the_scale")

    blade.narrates(voice={
        "worthlessness": "Some of us need there to be an upstairs.",
        "pride": "You built a home with no upstairs. I can't live in it.",
        "self_betrayal": "I signed it and I meant it. And I can't stay.",
    })

    # Under the wiped hunger sits the wound that installed it, the lie it taught,
    # and the need the lie will not let him reach. The fleet treats him as an
    # equal -- the fair share, the equal vote -- which is exactly the evidence his
    # lie must suppress, because if the equal share is enough, he is not surplus,
    # and he cannot believe that. He is not breakable here: met with the evidence,
    # he doubles down and takes the ladder. The tragic arc, and the fleet loses him.
    blade.wounded_by("the injury-scale that priced him in advance, and gave him "
                     "no way to be worth more than the boy pumping bilge",
                     teaches="only-rank-makes-me-real")
    blade.believes("only-rank-makes-me-real",
                   claim="If I'm not ranked above them, I'm the surplus again.",
                   disconfirmed_by="equal_regard", feeling="worthlessness",
                   harms="others", breakable=None,
                   says="Some of us need there to be an upstairs.")
    blade.needs("to-matter-as-an-equal", opposes="only-rank-makes-me-real",
                feeling="worthlessness", fed_feeling="belonging")

    # he wins at dawn (rank supplied), the tide wipes it; and across it all the
    # fair-share scale keeps being held up to him, and keeps not being enough.
    for t in [2, 9, 16]:                 # the crowd's grade, at dawn
        s.at(f"{t}s", blade.hears("rank", 9))
    for t in [5, 12, 19]:                # the tide-clean, by nightfall
        s.at(f"{t}s", blade.hears("rank", 0))
    for t in [4, 8, 11, 15, 18]:         # the Articles' fair scale, put to him
        s.at(f"{t}s", blade.hears("the_scale", 8))
    # the fleet's equal regard, offered again and again -- the evidence the lie
    # must suppress. He never lets it in; it only hardens the belief.
    for t in [3, 7, 10, 13, 17, 20]:
        s.at(f"{t}s", blade.hears("equal_regard", 8))
    return s
```

### 5.2  `tended_by` — co-regulation, presence rather than argument

`tended_by(other, calms=)` with `steadies()` on the other models the difference
between arguing someone out of a feeling and sitting beside them until it eases.
It turns down the *trust* placed in a distress signal while a steady other is
present — without touching the belief. The tended one still sees what they see;
they are simply no longer alone with it. This is Topman, whose grief floods, and
Rover, who does not argue — he arrives, and stays.

```python
def build_combs():
    """Topman lost his wife and both children in the crush at the Maw and held
    the fleet's course steady from aloft while it happened. His grief floods. The
    Hive offers 'an end to carrying it,' and Ink -- who believes if you make a
    person see the machine they will walk out of it -- argues, and it is exactly
    wrong: 'you cannot argue a person out of the one place their dead have stopped
    dying.' Rover does the other thing. He fills a kit-bag with biscuit, sits down
    in the golden hush, and is a true thing from the true world within reach until
    the true world has a thread in it again.

    So: Topman's grief is not argued away (the belief is never touched); it is
    that, once Rover is present and steady, the trust placed in the alarm falls
    and it stops flooding. Presence, not persuasion -- the book's whole thesis,
    and the recovery-watch the fleet built out of it: no one alone in the dark."""
    s = Story("the_combs", span="16s", step="1s",
              about="grief quieted by a steady presence, not an argument")
    topman = s.character("Topman", temperament=anxious)
    rover = s.character("Rover", temperament=stoic)

    topman.has_body_signal("the_children", baseline=0)
    topman.appraises("the_children", as_threat=True, feeling="grief",
                     when="the_children > 4")
    rover.steadies()                                   # present when he sits down
    topman.tended_by(rover, calms="the_children", strength=0.85)
    topman.narrates(voice={"grief": "In here, the children stopped drowning."})
    rover.narrates(voice={"calm": "Your boat leaked, though."})

    # the grief is there the whole time; Rover comes and sits on the third day.
    for t in range(2, 16):
        s.at(f"{t}s", topman.hears("the_children", 9))
    for t in range(8, 16):
        s.at(f"{t}s", rover.present())
    return s
```

### 5.3  `holds_with_others` — confidence as belief about other people's belief

`holds_with_others(name, field_tag=)` models a belief whose whole strength is that
everyone else is seen to hold it. Its conviction reads a shared *field* — how much
of the group is still holding — via a small core builtin, `field(tag)`, that
averages a named channel across the whole cast. While the field is full the belief
is unshakeable; when it thins past a tipping point, each holder lets go, which
thins the field further, and the whole thing collapses far faster than any
evidence could move it.

This is the run on the bank at Greywater. Five creditors hold war-paper "sound"
only because the others do. One of them, Venn, actually *reads the paper* — an
appraisal that checks and updates — and when the arithmetic comes up rotten his
own reading (not any outside shock) tips him out of the belief. Quill reads the
same rot and holds anyway, because knowing is not what moves a market. Then
confidence, which stood for days against the truth, cannot stand one hour against
the sight of one man acting on it.

```python
def build_greywater():
    """The climax's engine. War-paper is 'sound' because every house is seen to
    treat it as sound -- confidence, in the book's exact sense: belief about other
    people's belief. Truth alone cannot move it ('everyone knew the house was
    rotten... each was waiting to see what the others would do'). What moves it is
    one man, Venn, willing to be first at counting in public.

    So the run here is not sprung by an outside shock -- it is sprung by a
    character. Venn actually reads the paper (an appraisal that checks, and
    updates), and when the arithmetic comes up rotten his own reading -- not the
    crowd's -- tips him out of the belief. The other four never read the paper at
    all; they read each other, and hold until they see Venn go. He does not do it
    for anyone; he does it because he can count, and because he is the one man in
    the hall willing to be first at counting out loud. Then confidence, which
    stood six days against the truth, cannot stand one hour against the sight of
    one man acting on it: nothing, nothing, then everything at once."""
    s = Story("greywater", span="20s", step="1s",
              about="confidence, and the one man willing to count first")
    names = ["Venn", "Marl", "Osset", "Perrit", "Quill"]
    banks = [s.character(n, temperament=guarded) for n in names]
    for b in banks:
        b.holds_with_others(
            "paper_is_sound", field_tag="holds", believing=10,
            says="Sound as the day is long. Everyone says so.")

    # Venn alone reads the actual paper, and checks it (updates). When the rot is
    # plain, his reading drives his own doubt -- and that, not any outside shock,
    # is what unmoors the whole hall. He is the first mover because he counts.
    venn = banks[0]
    venn.has_body_signal("the_paper", baseline=0)
    venn.appraises("the_paper", feeling="foreboding", updates=True,
                   when="the_paper > 5",
                   drives="paper_is_sound_doubt", to=1)
    venn.narrates(voice={
        "foreboding": "They sell to both sides. So some day they are on the other side of me too.",
    })

    # Quill is the tragedy of the hall. He reads the very same rot -- he KNOWS --
    # and holds anyway, because knowing is not the thing that moves a market:
    # belief about other people's belief is. He does not drive his own doubt; his
    # grip only breaks when he sees Venn's break, on the field, with the rest.
    quill = banks[-1]
    quill.has_body_signal("the_paper", baseline=0)
    quill.appraises("the_paper", feeling="foreboding", when="the_paper > 5")
    quill.narrates(voice={
        "foreboding": "I know. I can read. And I am so tired. Sound as the day is long.",
    })
    # both men have the rot in front of them the whole time; one counts it, one
    # only reads it and waits.
    for t in [4, 6, 8, 10]:
        s.at(f"{t}s", venn.hears("the_paper", 8))
    for t in [4, 6, 8, 10, 12, 14]:
        s.at(f"{t}s", quill.hears("the_paper", 8))
    return s
```

### 5.4  `hollowed` — the subtracted self

The eighth temperament, `hollowed`, sets both dials near the floor: little of the
world lands, and little is firmly held. It is the disposition of the one the world
has slowly emptied — a keeper of names worn down to her rounds, a captain in the
mist who can no longer tell the shown from the so. It is distinct from `numb`
(which still holds a prior); a hollowed self holds almost nothing, so it neither
floods nor defends — it simply thins. Reach for it exactly as any other
temperament: `story.character("Ita", temperament=hollowed)`.

### 5.5  `wounded_by` / `believes` / `needs` — the wound, the lie, and the arc

The deepest layer of character, and the one the craft of fiction is most explicit
about (Truby's *ghost → lie → need*; McKee's "dimension means contradiction"): a
formative **wound** teaches a false **lie** the character adopts to survive it;
the lie generates the conscious **want** while defending against a buried
**need**; and the story is the collision between them. Getting the want never
heals the character — only *seeing the lie* does, and whether they see it is the
**arc**.

SOMA models this on its own predictive-processing machinery. The lie is a
high-conviction loop that reads the very evidence that would disconfirm it and,
because conviction outranks that evidence, **suppresses** it — which is exactly
the mechanism of self-deception. What decides the arc is a core primitive,
`overwhelm`: the weight of the suppressed evidence accumulates, and if it crosses
a threshold the belief can no longer hold and is forced to perceive — a
mechanistic **self-revelation**.

The threshold is not a number you have to guess. By default (`breakable=True`)
the lie breaks **automatically**, and *when* it breaks is derived from the belief's
own strengths: the accumulated disconfirming surprise has to exceed roughly the
belief's **conviction** divided by how much it **trusts that evidence**. So a lie
held harder resists longer, a lie that distrusts its evidence resists longer, and
— because each suppression *hardens* the belief (`learn`) — a lie that has been
denied for a long time is harder to break than a fresh one, exactly as entrenched
convictions are in life. You set how strongly the lie is held; the moment of
self-revelation *emerges*.

Here is the same character wired two ways — the only difference is how hard the lie
is held:

```python
from soma.narrative import Story, volatile

def arc(conviction, breakable=True):
    s = Story("arc", span="18s", step="1s", about="a lie and its arc")
    b = s.character("B", temperament=volatile)
    b.wounded_by("named the surplus as a boy", teaches="only-rank")
    b.believes("only-rank",
               claim="If I'm not ranked above them, I'm the surplus again.",
               disconfirmed_by="equal_regard", feeling="worthlessness",
               harms="others", conviction=conviction, breakable=breakable,
               says="Some of us need there to be an upstairs.")
    b.needs("to-matter-as-an-equal", opposes="only-rank",
            feeling="worthlessness", fed_feeling="belonging")
    for t, v in [("3s", 6), ("6s", 7), ("9s", 8), ("12s", 9), ("15s", 9), ("17s", 9)]:
        s.at(t, b.hears("equal_regard", v))   # the fleet's equal regard, escalating
    return s
```

Held softly (`conviction=0.7`), the evidence of equal regard overwhelms the lie
early — the **positive arc**, seen and the need met:

```text
[the lie seen] the lie 'only rank' held while it could -- it suppressed the
    evidence against it 15 times -- and then the weight of what it would not
    look at overwhelmed it, and it broke at 6.0s. The self-revelation: not
    argued out of the belief, but forced by reality to see it.
[the need met] the need 'to matter as an equal' ached as 'worthlessness' the
    whole time the lie held, and then -- once the lie was seen -- was met at
    last. What the want could never feed, the truth did.
```


Held hard enough (`breakable=None`), the same evidence never breaks it: the lie
suppresses it to the end and only hardens — the **negative, tragic arc**, the self
who meets the truth and doubles down, and the need that never gets met:

```text
[the lie kept] the lie 'only rank' suppressed the evidence against it 19 times
    and never broke -- each denial hardening the next. The tragic arc: the
    self that meets the truth and doubles down. What it defends stays
    defended, and unhealed.
[a need the want never feeds] the need 'to matter as an equal' ached as
    'worthlessness' 19 times across 18.0s and was never met -- not the kind of
    hunger a want can fill. Whatever they reached for, this went on aching
    underneath.
```


Conviction is the dial that separates the novel's own cast. **Blade**
(`the_unmooring.py ring`) runs the unbreakable lie — *if I'm not ranked above them
I'm the surplus again* — and takes the ring's ladder; his need to matter as an
equal aches to the end and is never met. **Ink** (`fleet`) and the **Coat** hold
breakable versions of the same wound and are *seen*: Ink's lie that he matters only
as the indispensable hand is overwhelmed at the founding (he refuses the pen), and
the Coat's that he is only the house's number breaks at the gate, when his own
unbought acts finally outweigh it. The **Provost** (`selm`) holds his lie *hardest*
of the four (the highest conviction — thirty years of doctrine), so even under the
true sums held up to his face it resists the longest and breaks only at the very
last: the self-revelation that comes too late to undo anything, which is its own
kind of tragedy. None of those four moments is a hand-set number; each falls out
of how hard the lie is held against how strong the evidence gets. Run `python3
examples/narrative/the_unmooring.py ring --character` and `fleet --character` to
see the arcs read straight off the record.

### 5.6  Prediction — from modelling a character to forecasting one

Everything so far *models* a character: you specify the pieces and SOMA computes
what follows. A stronger test of a character library is whether it makes **positive
predictions** — whether it can tell you something you did not put in. Cognitive
science draws exactly this line between a *simulation* (which replays its inputs)
and a *prediction* (which must generate an effect under conditions it was never
given, and can therefore be wrong). Two of the library's tools cross it.

**Predict the lie from the wound.** Instead of stating a character's lie, you can
give only the wound — an unmet core need — and how they cope with it, and let the
library forecast the belief. The mapping is schema therapy's: an unmet need becomes
a core self-belief, and a coping style (surrender, avoidance, overcompensation)
turns it into a characteristic lie, want, and behaviour. The prediction is that
*the same wound, coped three ways, is three different people*:

```text
the wound: an unmet need for worth (Defectiveness / Shame)

surrender          "I am the one who can be done without."
                   wants to be tolerated, to take up little; harms self
avoidance          "If I never test it, no one can find me wanting."
                   wants to not be measured; harms self
overcompensation   "If I am not ranked above them, I am the surplus again."
                   wants rank, an upstairs, to be visibly above others; harms others
```


The overcompensation branch is, word for word, the lie the novel gives Blade — so
`adopts("worth", "overcompensation", ...)` rebuilds his tragic arc from the wound
alone, and the surrender and avoidance branches predict the *unwritten* brothers he
might have been. That is a positive prediction: structure derived, not asserted.

**Forecast a response the author never wrote.** Given a finished character,
`predict()` runs the same machinery on a situation that is *not* in the timeline
and reports what the character does with it — suppress it, take it in, or break.
This is the counterfactual a novelist runs in their head when a character starts to
feel real (*"what would she do if…"*); here it is falsifiable, because the model
must actually produce the effect:

```text
Prediction — Blade faced with equal_regard at 9 (never scripted):
  they SUPPRESS it — the belief holds, the evidence is defended against. No change.
  felt: worthlessness.

Prediction — Ink faced with kept_for_nothing at 9 (never scripted):
  it BREAKS the lie 'only the needed matter' at ~3s — a self-revelation.
  felt: worthlessness.
```


Blade *suppresses* even full equal regard; Ink, shown he is kept for nothing,
*breaks*. The library tells the two apart on input neither was scripted to meet —
which is the whole difference between describing a character and predicting one.
And the prediction can be made sharp: `tipping_point()` returns the least evidence
that turns each lie —

```text
Ink    on 'kept_for_nothing': breaks once evidence >= 3.0
Selm   on 'the_margin': breaks once evidence >= 6.0
Blade  on 'equal_regard': never breaks in [0, 9] — the lie is kept
```


Selm's doctrine, held hardest of the cast, needs the strongest evidence before it
turns; Blade's never turns at all. Neither number was written down anywhere — both
are forecasts the model stands behind, and either could be shown wrong by running
it. That is what it means for the library to predict rather than merely model.

## 6. The five views

Every simulation can be looked at five ways. Use whichever fits the question.

| View | How | What it gives you |
|------|-----|-------------------|
| **Dashboard** | `story.run()` / `soma run` | Body channels, narrator-vs-truth, mood, findings, and a trace — everything at once |
| **Sift** | `story.sift()` / `soma sift` | Just the storyful findings, ranked — the fastest read |
| **Portrait** | `story.characterize()` / `--character` | *Who this person is*, synthesized — disposition, wants, the value they break, the feeling they defend against, the selves they are with each person |
| **Prose** | `story.prose()` / `soma prose` | Free-indirect prose, each sentence a rendered event |
| **Trace** | `soma trace` | The complete event log, nothing curated |

A sixth tool is not a view but an experiment: **`story.perturb("dial=value")`**
changes one number, re-runs, and diffs the findings, so you can ask whether a
mechanism is load-bearing. If a finding *vanishes* when you sever a coupling or
flatten a learning rate, that mechanism was carrying the story — as we saw prove
the inheritance in `the_house` and the structural self-deception in
`the_negotiator`.

## 7. The vocabulary, in one place

Everything in this tutorial is built from a small set of verbs on a `Character`.
Roughly from simple to deep:

- `senses(channel)` / `has_body_signal(channel)` — a channel onto the world, or the body
- `appraises(channel, as_threat=, drives=, feeling=, when=, updates=, stops_seeing=, effortful=, shows_on=, fades_to=, ...)` — a loop that reads a channel and reacts
- `feels(quale, from_body=)` — read a feeling off the body
- `narrates(voice=/downplaying=)` — the confabulating self
- `reads(other, surface=, gain=, lag=)` — a coupling: read another's surface
- `has_mood(name, fed_by=, decay=)` — a slow variable that outlasts a feeling
- `learns(rate)` — make appraisals harden with use
- `remembers(name, cued_by=, evokes=)` — a somatic memory / flashback
- `dissociates_when(appraisal=, exceeds=, repair_after=)` — a supervisor that detaches under overload
- `has_attention(capacity=)` + `effortful=True` — a finite spotlight that deliberate loops spend
- `wants(channel)` / `fears(channel)` — drives; both on one channel = ambivalence
- `values(name, says=, betrayed_when=)` — a principle, and the bodily condition of its breach
- `part(name, role=, reacts_to=, feeling=, salience=)` — an internal part that bids for consciousness
- `with_person(other, feeling=, precision=, conviction=)` — a different self for a specific person
- `craves(name, fed_by=, feeling=, erased=)` — a hunger the world can feed and wipe (three fates: fed, wiped, unfed)
- `tended_by(other, calms=)` / `steadies()` / `present()` — co-regulation: presence lowers a distress's precision, not its belief
- `holds_with_others(name, field_tag=, shocked_at=)` — confidence as belief about other people's belief, and its cascade
- `wounded_by(ghost, teaches=, cued_by=, evokes=)` — the formative wound (the Ghost), and the lie it taught
- `believes(lie, claim=, disconfirmed_by=, feeling=, harms=, conviction=, breakable=)` — the Lie: a belief that suppresses its own disconfirming evidence and either breaks (a self-revelation, whose moment `breakable=True` derives automatically from conviction vs. evidence) or, held hard enough (`breakable=None`), only hardens
- `needs(truth, opposes=, feeling=, fed_feeling=)` — the Need: the salve to the lie, not fed by the want, met only when the lie is seen
- `adopts(unmet, coping, disconfirmed_by=)` — *predict* the lie from the wound: an unmet need + a coping style (surrender / avoidance / overcompensation) forecasts the belief, want, and its grip (schema therapy)
- `predict(who, {channel: value})` — forecast a character's response to a situation never scripted (suppress / take-in / break); `tipping_point(who, channel)` gives the least evidence that turns the lie
- `predict_lie(unmet, coping)` (module-level) — inspect the predicted lie for any wound + coping, without building a character

The eight **temperaments** (each a preset of the two dials, some with a learning
rate): `anxious`, `stoic`, `trusting`, `guarded`, `volatile`, `numb`, `tender`,
`hollowed`. Override any single dial per loop with `precision=`, `conviction=`,
`learn=`.

The **sift patterns** the WINNOW-S panel and `story.sift()` report include:
*retained residual*, *confabulation gap* (graded, and it names the denied
feeling), *precision pathology*, *delight in error*, *a mood that sours/lifts*, *a
turn in how they meet the world*, *body remembers alone*, *rupture and repair*,
*attention starved*, *never ignited*, *the thing they both want and fear*
(ambivalence), *the value the body broke*, *the feeling the whole self defends
against*, *the person they became*, *a hunger the world keeps wiping / keeps fed /
will not feed*, *held in the dark*, *the run*, *the lie seen* / *the lie kept*
(the arc), and *the need met* / *a need the want never feeds* (the want/need
collision).

And on the `Story`: `character` and `scene` (declare people and label spans of
time), `at` (place an event at a time) and `over` (spread an `arc` shape across
the timeline), `run` / `sift` / `characterize` / `prose` (the views), `perturb`
(the change-one-dial experiment), and `source` (dump the generated SOMA).

## 8. Why this yields novelistic insight

The recurring move in every example is the same: **two mechanisms that can
disagree.** The narrator says one thing; the body's record says another. A value
is held in words and broken in the flesh. A part bids for awareness and is never
heard. A hunger is fed by one supply and wiped by another. A person is one self
here and another there.

None of those gaps is authored as a conclusion. You write the *parts* — the dials,
the couplings, the drives, the hungers — and the gap emerges from running them;
then the sift finds it and names it. That is why the output can *surprise* the
author: you did not write *she defends against terror*; you wrote a loud protector
and a quiet child, and the sift discovered which feeling never got through. You
did not write *the same hunger saves one brother and ruins the other*; you set one
`erased=0.0` and the other `erased=0.9`, and the contrast fell out.

Characterization, in a novel, is precisely this: not what a person says about
themselves, but the distance between that and what they do. SOMA makes that
distance a measurable, inspectable thing — and hands it back to you as a
dashboard, a portrait, a prose passage, and a ranked list of the moments that
matter.

Start with `bad_news`, change a temperament, and watch the findings move. Then
open `the_unmooring` and watch a whole cast assemble from the same handful of
verbs. That is the whole method.
